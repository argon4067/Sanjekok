# 개요
- 산재 예방·병원·통계 정보가 분산되어 있어 발생하는 정보 접근성 문제를 해결하기 위해, 
한 곳에서 산재 위험도 분석과 예방·대응 정보를 통합 제공하는 서비스를 기획했습니다.
